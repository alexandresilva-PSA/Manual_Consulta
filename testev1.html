Visão Geral

1.1 O que é a Consulta EFD Contribuições?

A ferramenta Consulta EFD Contribuições é um ambiente especializado para auditar, visualizar e extrair dados de arquivos SPED (PIS/COFINS). Ela elimina a necessidade de softwares externos pesados (como o PVA da Receita) para consultas rápidas e exportações de dados específicos.

1.2 Funcionalidades Chave

  • Repositório Centralizado: Acesso a todo o histórico de arquivos SPED dos clientes.
  • Busca Inteligente: Filtragem por período com detecção automática de intervalos (arquivos que intersectam o mês selecionado).
  • Exportação Granular: Capacidade de extrair apenas registros específicos (ex: somente notas fiscais C100) para Excel.
  • Perfis Personalizados: Salve suas configurações de exportação (ex: "Auditoria de Crédito") para reutilizar futuramente.
  • Análise Web: Visualização de dados, totais e itens diretamente no navegador, sem download.

Filtros de Busca

3.1 Configurando os Filtros

A busca é composta por quatro critérios principais. O preenchimento segue uma ordem lógica para facilitar a localização dos arquivos.

Visão geral dos filtros de busca

3.2 Seleção de Cliente e Contribuinte

Primeiro, selecione o Cliente. O sistema carregará automaticamente a lista de contribuintes vinculados.

auto_awesome

Dica de Produtividade: Se o cliente possuir apenas um contribuinte cadastrado (matriz única), o sistema o selecionará automaticamente para você.

O CNPJ é exibido ao lado da Razão Social para conferência

3.3 Definição de Período (Intervalo)

Defina a Data Início e Data Fim. O sistema buscará todos os arquivos cuja vigência (DT_INI a DT_FIN do arquivo) coincida, mesmo que parcialmente, com o período selecionado.

Seletor de Mês/Ano simplificado

3.4 Executar e Limpar

Clique em Buscar Arquivos para carregar a lista. Caso queira reiniciar a pesquisa, utilize o botão Limpar Filtros (ícone de borracha) para resetar todos os campos.

Downloads (TXT/ZIP)

5.1 Download Individual (TXT)

Para baixar o arquivo original SPED (formato texto), clique no primeiro ícone da coluna "Ações" (seta para baixo).

Feedback visual de sucesso no download

5.2 Download em Massa (ZIP)

Para baixar todos os arquivos listados de uma só vez, utilize o botão Baixar Todos no topo da tabela.

info

O sistema verificará a integridade de todos os arquivos. Caso algum arquivo antigo não esteja mais no armazenamento, você receberá um aviso detalhado informando quantos arquivos foram recuperados com sucesso.

O sistema gera um pacote ZIP automaticamente

6

Seleção de Registros

6.1 O Modal de Exportação

A ferramenta de exportação permite converter dados complexos do SPED para planilhas Excel organizadas. Clique no ícone do meio (Planilha) na coluna de ações.

Interface principal de seleção de dados

6.2 Navegação em Árvore (Blocos)

Os registros são agrupados por Blocos (0, A, C, D, F, M, 1, 9). Clique na seta ou no nome do bloco para expandir e ver os registros contidos.

Visualização hierárquica dos blocos

6.3 Modos de Seleção

  • Seleção Global: Clique em "Selecionar Todos" no topo para marcar o arquivo inteiro.
  • Seleção por Bloco: Clique no checkbox ao lado do nome do bloco (ex: Bloco C) para marcar todos os seus registros filhos.
  • Seleção Individual: Marque registros específicos (ex: apenas C100 e C170) para relatórios mais leves e focados.

O contador (ex: 7/9) indica seleção parcial do bloco

7

Gerenciamento de Perfis

7.1 Por que usar Perfis?

Perfis permitem salvar sua seleção de registros. Se você realiza auditorias recorrentes (ex: conferência de alíquotas de PIS/COFINS), pode salvar um perfil contendo apenas os registros relevantes (C170, M210, M610) e carregá-lo com um clique no futuro.

7.2 Criando um Novo Perfil

Após selecionar os registros desejados, clique no botão + (Mais) ao lado do seletor de perfis.

Defina um nome (ex: "Auditoria Mensal") e, opcionalmente, marque a caixa Definir como padrão para que este perfil seja pré-carregado sempre que você abrir o modal.

7.3 Carregando e Alternando Perfis

Utilize o menu dropdown "Carregar Perfil" para alternar instantaneamente entre suas configurações salvas. O perfil marcado com uma estrela (★) é o seu padrão atual.

Lista de perfis personalizados

7.4 Editando e Excluindo

No dropdown de seleção, ao passar o mouse sobre um perfil, um ícone de Lixeira aparecerá. Clique nele para remover perfis obsoletos.

8

Geração e Status

8.1 Processamento Inteligente

Ao clicar em Gerar Relatório, o sistema inicia o processamento. Diferente de downloads comuns, esta ferramenta processa o arquivo no servidor para não travar seu navegador.

8.2 Status da Exportação

O botão mudará para Processando.... Você pode acompanhar o progresso através das mensagens de status no rodapé do modal.

bolt

Cache Inteligente: Se você ou alguém da sua equipe já gerou este mesmo relatório recentemente, o sistema detectará o arquivo pronto e o download iniciará instantaneamente, pulando a etapa de processamento.

8.3 Conclusão

Assim que finalizado, o download do Excel iniciará automaticamente e uma mensagem de sucesso será exibida.

9

Interface de Análise

9.1 Visão Geral da Análise

O modo de análise permite auditar o conteúdo do arquivo sem precisar baixá-lo. Clique no botão Analisar (ícone de gráfico) na lista de arquivos.

9.2 Dashboard do Arquivo

O modal de análise abre em tela cheia. No cabeçalho superior, você tem acesso imediato aos totais consolidados do arquivo:

  • Identificação: Nome do arquivo e indicador de Retificadora.
  • Totais Fiscais: Somatório de PIS Devido e COFINS Devido extraídos do Bloco M.

Destaque para os totais no canto superior direito

10

Navegação Detalhada

10.1 Árvore de Navegação

A barra lateral esquerda exibe a estrutura hierárquica do arquivo. Blocos que contêm dados estarão disponíveis para clique. Selecione um registro (ex: C100) para carregar seus dados na tabela principal.

10.2 Tabela Dinâmica e Paginação

Os dados são carregados de forma paginada para garantir performance, mesmo em arquivos com milhares de notas.

  • Scroll Horizontal: Utilize a barra de rolagem inferior para ver colunas ocultas (como valores detalhados de impostos).
  • Formatação: Valores monetários, datas e CNPJs são formatados automaticamente para facilitar a leitura.
  • Paginação: Utilize os controles no rodapé (Anterior/Próximo) para navegar entre as páginas de registros.

Visualização tabular dos registros fiscais

Clique na linha para destacar o registro sob análise

Dicas de Produtividade

  • Economize Tempo: Crie um perfil padrão (ex: "Auditoria Básica") com os blocos 0, C e M selecionados. Assim, você não precisa remarcar as caixas toda vez que abrir a exportação.
  • Download ZIP: Ao baixar arquivos de um período longo (ex: 1 ano), prefira usar o botão "Baixar Todos" em vez de clicar um a um.
  • Verificação Rápida: Use a coluna "Crédito PIS/COFINS" na lista principal para identificar discrepâncias de valores antes mesmo de abrir o arquivo.
  • Conferência de Retificadoras: Se houver dois arquivos para o mesmo mês, verifique sempre o Badge (Original vs Retificadora) e a data de transmissão para usar a versão mais recente.
