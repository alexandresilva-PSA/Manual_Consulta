Visão Geral

1.1 O que é a Consulta EFD Contribuições?

A ferramenta de Consulta EFD Contribuições é um ambiente desenvolvido para simplificar a auditoria, extração e conferência de arquivos SPED (PIS/COFINS). Ela conecta diretamente à base de dados de arquivos transmitidos, oferecendo uma camada de inteligência que elimina a necessidade de softwares pesados como o PVA para tarefas cotidianas.

1.2 Funcionalidades Principais

  • Repositório Centralizado: Histórico completo de arquivos transmitidos.
  • Exportação Inteligente: Crie planilhas Excel apenas com os registros que você precisa (ex: C100, M200).
  • Perfis de Exportação: Salve configurações de colunas para usar repetidamente.
  • Análise Web: Visualize dados brutos formatados sem precisar baixar o TXT.
2

Acesso ao Sistema

2.1 Login na Área da Equipe

Para iniciar, acesse o portal da PSA Consultores. Na página inicial, localize e clique no botão Área da Equipe no cabeçalho.

Tela inicial do portal PSA Consultores

2.2 Seleção de Ambiente

No menu suspenso de seleção de áreas, escolha a opção Digital.

2.3 Autenticação

Insira seu e-mail corporativo e senha para acessar o ambiente seguro.

2.4 Navegação no Dashboard

Após o login, você verá o Dashboard da equipe Digital. No menu lateral esquerdo, sob a seção "Digital Dev", clique em EFD Contribuições.

Menu de ferramentas do desenvolvedor/auditor

3

Configuração de Filtros

3.1 Visão Geral dos Filtros

A tela inicial apresenta quatro critérios principais de busca. Todos devem ser preenchidos para garantir uma busca precisa.

3.2 Selecionando o Cliente

Clique no dropdown Cliente. A lista exibirá todos os clientes ativos vinculados à sua conta.

3.3 Selecionando o Contribuinte

Após selecionar o cliente, o campo Contribuinte será desbloqueado. O sistema exibe a Razão Social seguida do CNPJ.

auto_awesome

Se o cliente possuir apenas um contribuinte cadastrado, o sistema o selecionará automaticamente para você.

3.4 Definindo o Período

Utilize os seletores de Data Início e Data Fim. O calendário permite navegar entre anos e meses facilmente.

Seleção de período (Mês/Ano)

3.5 Executando a Busca

Com todos os campos preenchidos, o botão Buscar Arquivos ficará habilitado. Clique nele para consultar a base de dados.

4

Resultado e Listagem

4.1 Tabela de Arquivos

O sistema lista todos os arquivos que coincidem com o período selecionado. A tabela fornece uma visão rápida da "saúde" fiscal do período.

4.2 Detalhes das Colunas

  • Arquivo: Nome do arquivo e ID interno.
  • Período: Data de início e fim da competência do arquivo.
  • Tipo: Indica se é Original (Azul) ou Retificadora (Amarelo).
  • Crédito PIS/COFINS: Valores totais extraídos do Bloco M para conferência rápida.
5

Baixando Arquivos

5.1 Download Individual (TXT)

Para baixar o arquivo original transmitido à Receita, clique no botão de Download (ícone de seta) na coluna de Ações.

Uma notificação confirmará o início do download.

5.2 Download em Massa (ZIP)

Para baixar todo o histórico listado de uma vez, clique no botão Baixar Todos localizado no cabeçalho da tabela.

O sistema compilará todos os arquivos TXT em um único arquivo ZIP.

6

Configuração de Colunas

6.1 Acessando o Modal

A ferramenta de exportação permite converter dados do SPED para Excel. Clique no ícone de Planilha na coluna de ações.

Modal de seleção de registros

6.2 Selecionando Todos os Registros

Para uma exportação completa, clique na opção Selecionar Todos. Isso marcará todos os blocos disponíveis no arquivo.

6.3 Expandindo Blocos

Clique sobre o nome de um Bloco (ex: Bloco 0) para ver os registros contidos nele.

6.4 Seleção Granular

Você pode selecionar registros específicos clicando nos checkboxes individuais. O contador do bloco atualizará para mostrar quantos registros foram selecionados (ex: 7 de 9).

7

Criando e Salvando Perfis

7.1 Iniciando a Criação

Perfis economizam tempo salvando sua seleção de colunas. Após marcar os registros desejados, clique no botão + (Adicionar).

7.2 Definindo Nome e Padrão

Digite um nome descritivo para o perfil (ex: "Auditoria Créditos"). Se desejar que este perfil seja carregado sempre que você abrir a ferramenta, marque a opção Definir como perfil padrão.

7.3 Confirmando

Clique em Salvar Perfil. O sistema confirmará a criação e aplicará o perfil imediatamente.

8

Gerenciamento de Perfis

8.1 Carregando um Perfil

Para alternar entre configurações salvas, clique no dropdown de perfis. Sua lista de perfis personalizados aparecerá abaixo das opções padrão.

8.2 Identificando o Padrão

O perfil definido como padrão é marcado com uma estrela amarela. Isso indica que ele é carregado automaticamente ao abrir o modal.

8.3 Excluindo um Perfil

Para remover um perfil obsoleto, abra o dropdown e passe o mouse sobre o nome do perfil. Um ícone de Lixeira aparecerá à direita.

Ao clicar na lixeira, uma confirmação será solicitada para evitar exclusões acidentais.

9

Geração do Relatório

9.1 Iniciando o Processo

Com os registros selecionados (ou perfil carregado), clique no botão Gerar Relatório. O contador indica quantos tipos de registros serão processados.

9.2 Status de Processamento

O sistema iniciará a geração do arquivo Excel no servidor. O botão mudará para o estado de "Processando...".

9.3 Download Final

Assim que o processamento terminar, o download iniciará automaticamente e uma mensagem de sucesso será exibida.

10

Interface de Análise

10.1 Acesso à Análise

Para auditar o conteúdo sem baixar o arquivo, clique no botão Analisar (ícone de gráfico) na tabela de arquivos.

10.2 Visão Geral do Modal

A tela de análise abre em modo expandido. O cabeçalho exibe informações vitais como a Razão Social, Período e Tipo (Original/Retificadora).

11

Navegação Hierárquica

11.1 Árvore de Blocos

A barra lateral esquerda organiza os dados na estrutura do SPED. Clique em um bloco para expandi-lo.

11.2 Explorando Registros Fiscais

O Bloco C contém as Notas Fiscais. Ao expandi-lo, você verá registros como C100 (Nota) e C170 (Itens).

Você pode manter múltiplos blocos abertos simultaneamente para comparar estruturas.

12

Tabela Dinâmica

12.1 Visualização de Dados

Ao clicar em um registro (ex: C170), os dados são carregados na área principal.

12.2 Scroll Horizontal

Para registros extensos, utilize a barra de rolagem horizontal para ver colunas ocultas como valores de PIS/COFINS por item.

12.3 Seleção e Foco

Clique em qualquer linha para destacá-la. Isso ajuda a manter o foco durante a conferência de valores.

12.4 Paginação

O rodapé da tabela permite navegar entre páginas de registros, essencial para arquivos com milhares de itens.
