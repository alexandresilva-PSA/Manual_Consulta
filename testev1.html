Visão Geral

1.1 O que é a Consulta EFD Contribuições?

A ferramenta Consulta EFD Contribuições é um ambiente centralizado para gestão de arquivos SPED (PIS/COFINS). Ela permite que a equipe fiscal localize, baixe, audite e exporte dados de arquivos transmitidos de forma ágil, eliminando a dependência de processos manuais ou do PVA para consultas rápidas.

1.2 Funcionalidades Chave

  • Repositório Histórico: Acesso rápido a todos os arquivos processados por Cliente e CNPJ.
  • Download Flexível: Opções para baixar o arquivo original (TXT) ou pacotes em lote (ZIP).
  • Exportação Inteligente: Geração de Excel customizável, com opção de salvar perfis de colunas para uso recorrente.
  • Auditoria Web: Visualização detalhada dos blocos e registros sem necessidade de download.

Acesso ao Sistema

2.1 Login na Área da Equipe

Acesse o portal da PSA Consultores e clique em Área da Equipe no canto superior.

Portal inicial PSA

2.2 Seleção de Área

No menu suspenso de seleção de área, escolha a opção Digital.

2.3 Autenticação

Informe suas credenciais corporativas na tela de login segura.

2.4 Navegação

No Dashboard, acesse o menu lateral Digital Dev e clique em EFD Contribuições.

Acesso à ferramenta no menu lateral

3

Filtros de Busca

3.1 Configuração Inicial

A tela inicial apresenta os filtros vazios, aguardando a definição do escopo de pesquisa.

3.2 Seleção de Cliente e Contribuinte

Selecione o Cliente e, em seguida, o Contribuinte (CNPJ). O sistema possui inteligência para pré-selecionar o contribuinte caso o cliente possua apenas um cadastro ativo.

Seleção hierárquica de Cliente e CNPJ

3.3 Definição de Período

Utilize os seletores de mês/ano para definir o intervalo de busca (Início e Fim). O sistema buscará qualquer arquivo que tenha vigência dentro deste intervalo.

3.4 Execução

Com os filtros definidos, clique em Buscar Arquivos.

4

Lista de Resultados

4.1 Visualização de Arquivos

A tabela exibe os arquivos encontrados, destacando o período de competência, o tipo (Original ou Retificadora) e os valores de crédito apurados.

Lista de arquivos com indicadores visuais de status

5

Downloads (TXT/ZIP)

5.1 Download Individual

Clique no ícone de seta na coluna de ações para baixar o arquivo TXT original.

Notificação de sucesso

5.2 Download em Lote

Para baixar o histórico completo, use o botão Baixar Todos no topo da tabela.

O sistema gerará um arquivo ZIP contendo todos os períodos localizados.

6

Configuração de Colunas

6.1 Abertura do Modal

Clique no ícone de Planilha para iniciar a configuração de exportação personalizada.

Visão geral do modal de seleção de registros

6.2 Seleção de Dados

Você pode selecionar registros específicos expandindo os blocos. O sistema permite seleção granular (ex: apenas C100 e C170).

Indicador de seleção parcial de registros

6.3 Seleção em Massa

Para exportar o arquivo completo, utilize a opção "Selecionar Todos".

Todos os registros marcados para exportação

7

Gerenciamento de Perfis

7.1 Criação de Perfis

Após definir sua seleção de colunas, salve-a como um Perfil para reutilização. Clique no botão +.

7.2 Definição de Padrão

Você pode marcar um perfil como padrão para que ele seja carregado automaticamente nas próximas vezes.

7.3 Uso e Exclusão

Alterne entre perfis usando o menu suspenso. Perfis favoritos são marcados com estrela. Use o ícone de lixeira para remover perfis antigos.

8

Geração e Status

8.1 Processamento

Ao clicar em Gerar, o sistema processa o arquivo. Se o relatório já tiver sido gerado anteriormente, o download é instantâneo (cache).

9

Interface de Análise

9.1 Acesso à Análise

Clique no botão Analisar na lista de arquivos para abrir o visualizador web.

9.2 Visão Geral

O modal exibe o cabeçalho com totais fiscais (PIS/COFINS) e a estrutura de navegação lateral.

Visualização inicial do arquivo

10

Navegação e Detalhes

10.1 Navegação em Árvore

Explore a estrutura do arquivo expandindo os blocos na lateral esquerda. Isso permite acesso rápido a qualquer registro.

Visão completa da estrutura do arquivo

10.2 Tabela de Dados

Os dados do registro selecionado são carregados na tabela central. Você pode selecionar linhas para destacar informações importantes durante a auditoria.

10.3 Visualização Estendida

Para registros com muitas colunas, utilize a barra de rolagem horizontal para visualizar campos adicionais.

11

Dicas de Produtividade

  • Atalhos de Perfil: Crie perfis para tarefas rotineiras (ex: "Conferência C100") para agilizar a exportação.
  • Verificação Prévia: Use a coluna "Crédito PIS/COFINS" na lista de busca para detectar anomalias antes de abrir o arquivo.
  • Gestão de Versões: Sempre verifique o selo "Original" ou "Retificadora" para garantir que está auditando a versão válida.
