Sumário

1

Acesso e Login

1.1 Página Inicial

Ao acessar o endereço do sistema, você verá a tela inicial da PSA. Clique no ícone de perfil no canto superior direito.

Página inicial

1.2 Menu de Equipe

Selecione a opção correspondente à sua equipe para prosseguir para a área de login.

Menu de seleção de equipe

1.3 Tela de Login

Insira seu e-mail corporativo e senha para autenticação.

Tela de login

1.4 Seleção de Área Digital

Escolha a opção "Digital" para acessar as ferramentas fiscais.

Seleção de área Digital

1.5 Dashboard

Você será redirecionado para o Dashboard. No menu lateral, localize "Consulta EFD Contribuições".

Dashboard geral

2

Filtros de Busca

2.1 Visão Geral dos Filtros

A barra superior apresenta os filtros disponíveis: Cliente, Contribuinte e Período.

Barra de filtros vazia

2.2 Seleção de Cliente e Contribuinte

Selecione primeiro o cliente (grupo) e depois o contribuinte específico.

Seleção de cliente

Seleção de contribuinte

2.3 Filtros Preenchidos

Após definir datas e empresa, clique em "Filtrar" para buscar os arquivos.

Filtros aplicados

3

Lista de Arquivos e Ações Rápidas

3.1 Lista de Arquivos

Os arquivos encontrados são listados com data, CNPJ e ações disponíveis.

Lista de arquivos EFD

3.2 Download Unitário e em Lote

Você pode baixar o TXT original de um arquivo específico ou todos os listados em um único ZIP.

Opção de baixar XML/TXT

Confirmação de download em lote

4

Exportação Detalhada (Excel)

4.1 Acessando a Exportação

Clique no botão "Exportar" na linha do arquivo desejado para abrir as opções detalhadas.

Botão de exportação

4.2 Seleção de Blocos e Registros

No modal, você pode escolher quais blocos do SPED deseja incluir. É possível selecionar blocos inteiros ou registros específicos.

Modal de exportação

Expandindo Bloco 0

5

Gerenciamento de Perfis de Exportação

Para agilizar o trabalho, você pode salvar conjuntos de seleções como "Perfis".

5.1 Salvando um Novo Perfil

Após selecionar os registros desejados, clique no ícone de disquete. Dê um nome ao perfil e, opcionalmente, marque-o como padrão.

Diálogo de salvar perfil

Definindo como padrão

5.2 Acessando Perfis Salvos

Utilize o menu suspenso para alternar rapidamente entre diferentes configurações (ex: "Completo", "Apenas Bloco M").

Alternando perfis

5.3 Excluindo Perfis

Para remover um perfil, clique no ícone de lixeira ao lado do nome e confirme a exclusão.

Confirmação de exclusão

6

Gerando o Relatório

6.1 Processamento

Ao confirmar a exportação, o sistema processará os dados e gerará o arquivo Excel.

Processando exportação

Conclusão

7

Análise Detalhada (Modal)

7.1 Visão Geral do Modal

A ferramenta de Análise permite visualizar o conteúdo do EFD sem precisar baixar ou abrir no PVA. A tela é dividida em Sidebar (estrutura) e Tabela (dados).

Modal de análise

7.2 Navegação por Blocos

Use a árvore lateral para expandir Blocos (0, A, C, D, M...) e selecionar Registros específicos (ex: C170, M200).

Navegando na estrutura

7.3 Visualização de Tabelas

Os dados são exibidos em grid. Use o scroll horizontal para ver todas as colunas. É possível clicar nas células para focar.

Tabela de dados (C170)
